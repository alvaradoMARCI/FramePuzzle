# FramePuzzle

> "Arma tus recuerdos"

App Android que transforma fotografías personales en puzzles interactivos.

## Stack
- Kotlin 2.0.20 + Jetpack Compose
- Koin (DI sin reflection)
- Room 2.6.1 con migraciones
- Navigation Compose 2.8.1
- CameraX 1.3.4
- Coil 2.7.0
- minSdk 28 (Android 9)
