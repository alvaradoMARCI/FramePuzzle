## Próximos hitos

1. **Compilación real en Android Studio** y prueba en emulador API 28+.
2. **QA (sección 51)**: pruebas unitarias, UI, rendimiento, seguridad y regresión.
3. **Beta cerrada** con el creador y usuarios invitados.
4. **Publicación** como `1.0.0` una vez cumplido el criterio de finalización (sección 56).

## Evolución futura (sección 71)

Toda nueva función debe:

- Mejorar la experiencia.
- Mantener privacidad.
- Respetar identidad del producto.
- No convertir FramePuzzle en otro tipo de aplicación.
